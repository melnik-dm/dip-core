Для ссылок используется следующий формат:

- ссылка на раздел [№](section2);
- ссылка на текст преобразуется в ссылку на раздел 
[№](section1/subsection1/Intro.md);
- ссылка на рисунок или схему [№](section1/subsection1/Plantuml.plantuml);
- ссылка на таблицу [№](section1/subsection1/Table.table).

Переход по ссылкам осуществляется с зажатой клавишей CTRL и кликом на 
идентификатор внутри круглых скобок.
