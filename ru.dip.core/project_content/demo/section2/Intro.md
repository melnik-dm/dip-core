В рамках проекта может быть создано неограниченное количество форм. Структура
формы описывается в формате xml. Формы определяются их расширением.

Место применение форм ограничивается лишь
фантазией автора документа. Формы могут быть использованы для требований (как
системных, так и ТВУ), для описания правил (стандартов на проектирование и
кодирование) и для описания любых других данных имеющих одинаковую форму.
